What is Responsive Web Design?

Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones)

Setting the viewport
Difference

Special features of the viewing area

  • minimum-scale
  • maximum-scale
  • user-scalable

Principles

  • Do not use large items with a fixed width.
  • To correctly display content, don't limit it to a specific width of the viewing area.
  • Use CSS media queries to specify different styles for large and small screens.

Media Queries 


                 
				

                 @media (query) {
                      /* CSS Rules used when query matches */
                }
				

                 @media print {
                      /* print style sheets go here */
                }
                @import url(print.css) print;

Main attributes

Attribute Result
min-width The rule applies to the browser whose width is greater than the value specified in the request.
max-width The rule applies to the browser whose width is less than the value specified in the request.
min-height The rule applies to the browser whose height is greater than the value specified in the request.
max-height The rule applies to the browser whose height is less than the value specified in the request.
orientation=portrait The rule applies to the browser whose height is not less than its width.
orientation=landscape The rule applies to a browser that is wider than its height.

NOTE: min-width vs min-device-width

Relative units usage

Not recommended — fixed width 



  div.fullWidth {
  width: 320px;
  margin-left: auto;
  margin-right: auto;
}
recommended — responsive width 



div.fullWidth {
  width: 100%;
}
